Test pages for visualization.

To run page from like_pmc folder:
0. start wordcloud server
1. start http-server inside like_pmc folder
2. open the server address in the browser and navigate to the author folder
3. open html file 
It will generate word cloud without buttons, by reading page address.

To run page from preview folder:
0. start wordcloud server
1. open index.html in the browser 
2. paste orcid into the filed and click generate.
